# Rust Bangalore Docker image

A developer centric Rust language runtime created by Rust Bangalore community.

Notable features
* built on ArchLinux x86_64 (pritunl/archlinux)
* vim editor
* git client

Rust specific packages
* rust
* rust-docs
* cargo
* rustfmt
* rust-racer
* uncrustify
* rustup
